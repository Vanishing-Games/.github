## 🚀 Our Standard

Every project under the **Vanishing Games** umbrella adheres to a strict set of principles:
- **Modularity First:** Systems should be decoupled and highly reusable.
- **Developer Experience (DX):** Tooling should feel native and intuitive within the Unity Editor.
- **Performance-Oriented:** Clean code shouldn't come at the cost of frame time.
- **Documentation-Driven:** If a system isn't documented, it doesn't exist.
